# superFuck
<div>自动骂人工具</div>
<div><b>运行前请安装依赖库(pip install +库名)</b></div>
<ul>
   <li> PyQt5</li>
   <li> requests</li>
   <li> pynput</li>
   <li> pyperclip</li>
</ul>
双击 运行.pyw运行,开启后点击要发送的文本框，自动发送

<p>功能:</p>
    <p>   可选择夸人、骂人，句子可从文本文档导入，也可直接用骂人网站里的句子（发送速度取决于网站回复速度  建议速度设置为0.1s）</p>
